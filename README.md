# AI Exam Preparation Chatbot

An intelligent, AI-powered assistant to help students prepare for exams by uploading books, PDFs, and YouTube links. It summarizes study materials , analyse PYQs/Syllabus and answers queries using a Multi-Agent Retrieval-Augmented Generation (RAG) pipeline powered by OpenAI , LangChain, langGraph and ChromaDB.
